## 一、2024B优秀论文格式特征分析

### 1.1 整体结构

| 章节 | 2024B 范本 | 页数 | 含图 |
|------|-----------|------|------|
| 封面 | 参赛编号 + 选题 + 赛道 + 标题 | 半页 | 无 |
| 摘要 | 按问题逐段描述方法+结果 | 1页 | 无 |
| 关键词 | 5-8个术语 | 1行 | 无 |
| 目录 | 三级标题 + 页码 | 1页 | 无 |
| 一、问题背景与重述 | 1.1背景 1.2重述 | 1页 | 无 |
| 二、问题分析 | 2.1-2.5逐个分析 | 1-2页 | 无 |
| 三、模型假设 | 编号列表 | 半页 | 无 |
| 四、符号及变量说明 | 表格 | 1页 | 无 |
| **五、模型建立与求解** | **主章节，分5个小节** | **30+页** | **√ 所有图** |
| 六、模型的评价与改进 | 优缺点+改进 | 1-2页 | 无 |
| 七、模型的推广与应用 | 1-2段 | 半页 | 无 |
| 参考文献 | 编号列表 | 1页 | 无 |

### 1.2 图表嵌入方式

- 图紧跟首次引用的文字段落之后，不另起页面聚集
- 图必须有编号（如"图1"、"图2"）和标题
- 表必须有编号（如"表1"、"表2"）和标题
- **图表从不放在附录**，全部嵌入正文相关位置
- 本文字号：正文小四(12pt)，图表标题五号(10.5pt)

### 摘要（约800字，5段对应5题）

> 参考2024B的写法：每段以"针对问题X"开头，简述方法+关键结果+量化指标。

```
针对问题一，对150,573条订单记录和72,129条菜品明细数据进行预处理，
通过描述性统计发现餐厅日均282人就餐、客单价11.39元。采用Apriori
算法挖掘19条关联规则，其中"米饭+酱鸭→豆芽/木耳"提升度达8.78。

针对问题二，构建SARIMA(1,1,1)(1,1,1,7)、XGBoost和组合预测的
多模型框架。Walk-forward验证MAPE为15.5%。给出2025年5月19个
工作日的预测结果，含95%置信区间。

针对问题三，建立混合整数线性规划(MILP)模型...5天午餐方案全部
Optimal，预期利润703-763元/天，营养均衡度0.93-0.94。

针对问题四，设计贪心搜索+局部优化算法...给出10/15/20元三价位套餐，
营养均衡度均达0.96以上。

针对问题五，从备菜策略、菜品结构、套餐推广、数字化运营和营养ESG
五个维度提出系统性策略，所有建议均附数据依据。
```
