### 需求3: 可视化配色改为 Nature 期刊常用配色

**变更文件**: `config.py`（核心）、全部5个可视化模块

#### 配色方案对比

| 颜色用途 | 原配色（Hex） | Nature NPG 配色（Hex） |
|----------|--------------|----------------------|
| 主色调 | `#2E86AB`（浅蓝） | `#3C5488`（深蓝） |
| 副色调 | `#A23B72`（品红） | `#00A087`（青绿） |
| 强调色 | `#F18F01`（橙） | `#E64B35`（红） |
| 成功/正向 | `#73AB84`（暗绿） | `#4DBBD5`（浅蓝） |
| 警告 | `#F18F01`（橙） | `#F39B7F`（鲑鱼） |
| 危险/重要 | `#C73E1D`（暗红） | `#DC0000`（深红） |
| 午餐标识 | `#F18F01`（橙） | `#E64B35`（暖红） |
| 晚餐标识 | `#2E86AB`（浅蓝） | `#3C5488`（冷蓝） |

#### 新增配色元素

| 元素 | 说明 |
|------|------|
| `COLOR_CYCLE`（7色序列） | 多系列图表使用：深蓝→红→青绿→浅蓝→灰紫→金→浅绿 |
| 热力图 `cmap` | `'RdBu_r'` → `'coolwarm'`（Nature 常用） |
| 散点图 `cmap` | 保持 `'YlOrRd'`（色盲友好） |
| 保存 DPI | `150` → `300`（满足 Nature 投稿要求） |
| 扩展色 | 新增 `purple` `#8491B4`、`teal` `#91D1C2`、`brown` `#7E6148`、`beige` `#B09C85`、`gold` `#E69F00`、`grey` `#A0A0A0` |

#### 配色设计原则

1. **高对比度**: 不同数据系列具有明显的视觉区分度
2. **色盲友好**: 避免红绿搭配，优先使用蓝-橙等安全组合
3. **印刷友好**: CMYK色域内，适合黑白/灰度打印
4. **学术规范**: 颜色饱和度适中，不干扰数据表达

参考来源: [ggsci R package "npg" palette](https://cran.r-project.org/web/packages/ggsci/)

#### 代表性注释示例

```python
def mape_score(y_true, y_pred):
    """
    计算 MAPE (Mean Absolute Percentage Error, 平均绝对百分比误差)

    公式:
      MAPE = (1/n) * Σ|(y_i - ŷ_i) / y_i| × 100%

    特点:
    - 优点: 直观易理解，以百分比表示预测误差
    - 缺点: 当实际值为 0 时无定义，对小值敏感
    - 适用: 餐饮人数预测 (值通常 > 0)

    处理策略:
    - 跳过 y_true = 0 的样本 (停业日就餐人数为 0)
    - 同时跳过 NaN 值
    """
```
